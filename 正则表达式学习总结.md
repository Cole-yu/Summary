# 	正则表达式学习总结

### 标准数字的表达式
```
let reg = new RegExp("^([+-]?)(\\d+)(\\.\\d*)?$", "i");
// let reg = new RegExp(/^([+-]?)(\d+)(\.\d*)?$/, "i");
// let reg = /^([+-]?)(\d+)(\.\d*)?$/;

reg.test("123"); 	// true (纯整数)
reg.test('123.45'); // true (带小数位)
reg.test('123.');   // true (末尾只有小数点)
reg.test('0.');     // true (0+小数点)
reg.test('abc');    // false (非数字)
reg.test('.123');   // false (开头只有小数点，如需支持可看下方扩展)


// 支持 123、 123. 、 123.45、 .45 所有正数字格式
let reg = /^([+-]?)(\d*)(\.\d*)?$/;
```

### $1,$2...$99(子表达式的分组)
	子表达式匹配的字符串,涉及知识点:捕捉分组/非捕捉分组的区别
	捕捉分组会将匹配结果存储在内存中,通过$1,$2...$99可以再次获取分组的内容
	name = "#into";
	console.log(name.replace(/.*#(\w+).*/, "$1"));			//$1、$2、...、$99	与regexp中的第1到第99个子表达式相匹配的文本。注:是从1开始
	输出结果:into    //$1是模式中的子表达式匹配的字符串,即(\w+)匹配的结果into

### match
	var str = "classA((\s)+|$)";
	var regexp = new RegExp(str);
	"classAc classB classAB classA".match(regexp);

### 取反操作
```
	[^xyz]，一个反向字符集
	通过^和[]，进行匹配
	str = str.replace(/[^0-9a-zA-Z]/gi, ''); // 只支持输入数字，大小写字母
```

### 匹配 "x" 或者 "y"
```
	x|y
	/green|red/ 匹配 "green apple" 中的"green" 和 "red apple" 中的"red"
```
