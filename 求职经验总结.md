###	简历撰写注意事项
```
	尽量写成一条一条的形式，简单清晰
	例如：
		1.全日制本科毕业，计算机科学与技术专业
		2.3年web开发经验，熟悉HTML5+CSS3+JS编程，熟悉标签语义化，网页性能优化，SEO优化等
		3.熟悉常用JS库，如JQuery、Zepto；熟悉常用UI类库，如Bootstrap、Amaze UI
		3.熟悉前后端分离，有移动端H5实际开发经验
		4.熟悉AngularJS、Vue、Angular2，了解React等前端开发框架
		5.有Vue + Vue-Router + Axios + Webpack + Vuex 技术栈开发经验
		5.有Angular2 + Typescript + RxJS + Bootstrap + NodeJs 技术栈开发经验
		6.有AngularJS + SeaJS + Jquery + Echarts + Javaweb 技术栈开发经验
		6.熟悉Webpack，实践过grunt，gulp等任务执行器的配置和插件开发
		7.熟悉WebApp实践,包括原生与js交互，SPA单页面Vue/Angular应用
		9.了解后台开发nodejs
		10.喜欢钻研各种新技术，乐于交流，有强烈的产品荣誉感及归属感，有清晰的职业规划
		11.有较强的学习能力，有自己的 Github 账号
```
